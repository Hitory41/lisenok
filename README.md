# Лисёнок — Детский логопед в Москве

Сайт для детского логопеда Бранки, расположенного по адресу: Кировоградская ул., 7, Москва.

## 🦊 О проекте

Современный одностраничный сайт (landing page) для специалиста-логопеда. Включает:
- Информацию о специалисте
- Описание услуг
- Галерею с фотографиями
- Контактную информацию
- Интерактивную карту Яндекс

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **Tailwind CSS** — стилизация через CDN
- **JavaScript** — интерактивность (карусель, анимации)
- **SVG иконки** — векторная графика

## 🚀 Развёртывание

### Локальный запуск

Просто откройте файл `index.html` в браузере.

### Публикация на Vercel

1. Создайте репозиторий на GitHub
2. Загрузите файлы проекта:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/ВАШ_НИК/lisenok.git
   git push -u origin main
   ```

3. Подключите репозиторий в [Vercel](https://vercel.com)
4. Сайт автоматически опубликуется
